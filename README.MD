# Vue2.5 Demo

使用 vue2+vuex+vue-router+webpack4+element-ui+sass+axios 搭建一个基本的实战框架

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
